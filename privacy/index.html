Policy blueprint

The sections below map the exact data touchpoints in ClassSync today, including what comes from Google, what stays on-device, which features are optional, and what users can disable or delete.

01

1. What ClassSync does

ClassSync is a local-first academic organization app for Android. It helps users combine Google Classroom coursework, announcements, and materials, optional Gmail-based academic reminder discovery, manually entered tasks, reminders, planner views, widgets, exam planning, and study planning.

ClassSync does not currently use a custom backend to centrally store synced academic content.

02

2. Information we access

  • Google account email address and basic sign-in session details
  • Google Classroom read-only academic data such as courses, coursework, announcements, and materials
  • Optional Gmail read-only message content that matches academic sync rules
  • Manually entered tasks and planner data
  • Local app preferences such as theme, reminders, sync settings, digest settings, and onboarding state
  • Generated local exports such as CSV and JSON task files when triggered by the user
03

3. How we use information

  • To sign you in with Google
  • To sync Classroom academic items into local events and tasks
  • To optionally detect academic reminders or updates from Gmail
  • To generate planner, study, exam, widget, digest, and reminder views
  • To support local ML-assisted and rule-based event classification
  • To save your app configuration and local academic workspace state
04

4. Optional vs required access

  • Google Classroom sync is optional app functionality, but it is a primary feature when enabled.
  • Gmail sync is optional and can be turned off independently.
  • Notifications are optional, but disabling them prevents reminders and digest alerts from being delivered.
05

5. Sharing and third parties

ClassSync does not sell personal data and does not operate a separate app backend that stores synced academic content for users.

  • Google sign-in services for authentication
  • Google Classroom API for read-only academic sync
  • Gmail API for optional read-only academic reminder discovery

If you export tasks, the resulting file is created on your device for your own use and may be shared by you using device-level sharing tools.

06

6. Local storage and security

ClassSync stores synced tasks, academic events, reminder and digest preferences, planner and exam-related state, and theme and onboarding settings locally on the device.

  • Google API traffic uses HTTPS/TLS
  • Sign-in session details are stored in encrypted local preferences where supported
  • The app is designed to keep academic data local-first by default
07

7. Permissions

  • INTERNET for Google sign-in, Gmail, and Classroom services
  • ACCESS_NETWORK_STATE to manage sync behavior based on connectivity
  • POST_NOTIFICATIONS for reminders, digest notifications, and due-soon alerts

ClassSync does not currently request contacts, location, microphone, camera, SMS, call log, photo library, or background location permissions.

08

8. Machine learning features

ClassSync may use an on-device TensorFlow Lite model to help classify academic events into categories such as task required, due date task, announcement only, material only, exam or test information, submission instruction, and grade or feedback.

This classification is performed locally within the app runtime together with deterministic rule-based logic and fallback policies.

09

9. Retention and user controls

ClassSync retains locally stored data on the device until you delete specific tasks or local academic data, sign out and clear local academic state, clear app data, or uninstall the app.

  • Sign out of Google
  • Disable Gmail sync
  • Disable Classroom sync
  • Disable background sync
  • Disable notifications
  • Delete local academic data through available app flows
10

10. Children and policy changes

ClassSync is intended as a general student productivity app and is not designed as a child-directed service under Google Play Families policies unless explicitly declared otherwise in the Play Console.

We may update this Privacy Policy as the app changes. If publishing changes introduce new data handling, permissions, or sharing behavior, this policy will be updated before those changes are released.
